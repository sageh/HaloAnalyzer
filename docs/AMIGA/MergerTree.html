NAME

AMIGA::MergerTree - An object oriented tree structure representation of halo data.

SYNOPSIS

        use AMIGA::MergerTree;
        my $tree = AMIGA::MergerTree->new(\%my_tree_structure);

INITIALISATION

new ( TREE )
This is the constructor for a new AMIGA::MergerTree object. TREE is a reference to a hash table containing the recursive tree structure.

This structure should consist of a hash table with halo indexes pointing to a array references containing:

  • Array reference to the _halos data for the halo.

  • A hash table with progenitor halo indexes pointing to array references containing:
    • Array reference to the _halos data for the halo.

    • A hash table with the progenitor indexes of this halo pointing to identical array references as above, with the recursion continuing through the tree.

The tree structure of the MergerTree is so an unordered tree, that can be only walked depthwards, i.e. from root node towards leaf nodes.

METHODS

root_node ( )

Returns the first node in the raw tree structure, as described in INITIALISATION.

Arguments: -

Return value: The root node of the tree structure.

double_link ( )

Makes the whole tree doubly linked, adding to each node array a hash reference of the parent node. This allows walking the tree in both directions.

Note: This method alters the object that it was invoked for. The process is irreversible.

Arguments: -

Return value: -

level_view ( )

Returns a 2D array of the contents of the tree structure, with the first dimension being the depth level of the tree, and the second the nodes at that depth. Thus the number of columns at each row equals the number of nodes at that depth.

Arguments: -

Return value: A 2D array of tree contents.

branchings ( [NODE] )

Returns the total number of branchings that occur in the tree after the node given in NODE or after the root node, if called with no arguments.

Arguments: NODE -- reference to an array containing node information. See INITIALISATION.

Return value: Total number of branchings depthwards from the given node or from the root node.
