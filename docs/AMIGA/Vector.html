NAME

AMIGA::Vector - Object oriented module for vectors and vector operations.

SYNOPSIS

        use AMIGA::Vector;
        # Create vectors in 3D.
        my $x = AMIGA::Vector->new(1, 2, 3);
        my $y = AMIGA::Vector->new(-1, 0, 0);
        # Addition and substraction operations are overloaded
        # for ease of use.
        my $z = $x + $y; # $z is now [0, 2, 3]
        print "x - y: ", $x-$y, "\n"; # prints [2 2 3]
        print "-x: ", -$x, "\n"; # prints [-1 -2 -3]
        # They can be called explicitly also, and can be
        # combined
        $z->add($y);   # $z now [-1, 2, 3]
        $z->subtr($y); # $z back to [0, 2, 3]
        $z->add($y)->subtr($y); # leaves $z unchanged
        # Dimensions must match
        $z = AMIGA::Vector->new(1, 2, 3, 4);
        print "x + z", $x + $z, "\n"; # Causes an error!
        # Vector operations
        print $x->dot($y), "\n"; # prints -1
        print $x->length(), "\n"; # prints 3.74165738677394
        print sqrt($x->dot($x)), "\n"; # same as above
        print "dimension: ", $x->dim(), "\n"; # prints 3
        # The components can be retrieved as an array if needed
        my @components = $x->to_array();

INITIALISATION

new ( COMPONENTS )
This is a constructor for a new AMIGA::Vector object. COMPONENTS should be an array of the component values of the vector.

METHODS

copy ( )

Returns a copy of this vector.

Arguments: -

Return value: Copy of the vector for which this method was invoked.

mult ( REAL )

Multiplies this vector with the given scalar REAL. Note: Changes the vector for which the method was invoked.

Arguments: REAL -- a scalar value.

Return value: The resulting vector.

add ( VECTOR )

Adds the vector VECTOR to this vector, and then returns it. Note: Changes the vector for which the method was invoked.

Arguments: VECTOR -- an AMIGA::Vector object.

Return value: The resulting vector.

subtr ( VECTOR )

Subtracts the vector VECTOR from this vector, and then returns it. Note: Changes the vector for which the method was invoked.

Arguments: VECTOR -- an AMIGA::Vector object.

Return value: The resulting vector.

neg ( )

Changes sign of every component of this vector, and also returns it. Note: Changes the vector for which the method was invoked.

Arguments: -

Return value: The resulting vector.

dot ( VECTOR )

Calculates the dot product between this vector and the vector VECTOR given as argument.

Arguments: VECTOR -- an AMIGA::Vector object.

Return value: Dot product of the two vectors.

length ( )

Calculates the length of this vector and returns it.

Arguments: -

Return value: Length of the vector.

to_str ( )

Renders the vector into a Perl string. This is done by printing the component values sequentially separated by whitespace inside enclosing square brackets. Thus a vector with components 2.1, 3.2, 4.3 renders into [2.1 3.2 4.3].

Arguments: -

Return value: Vector string representation.

dim ( )

Returns the dimension of the vector.

Arguments: -

Return value: Dimension of the vector.

to_array ( )

Returns the components of the vector as a Perl array.

Arguments: -

Return value: Vector components as an array.
