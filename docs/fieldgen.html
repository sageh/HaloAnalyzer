NAME

fieldgen.c -- A program for generating 3 dimensional random binomial fields suitable for using with the correlator and compcorr programs. The program uses the Mersenne Twister algorithm for fast high quality pseudorandom number generation.

SYNOPSIS

        # Generate a 200000 point field in output.dat, distributed between
        # (0, 0, 0) and (10, 10, 10), and by using the Mersenne Twister seeded
        # with the number 31337
        ./fieldgen output.dat 31337 10 200000

ARGUMENTS

The following arguments must be given, in this order: OUT SEED BOX POINTS.

  • OUT -- Filename where the generated data should be stored.

  • SEED -- A seed number for the Mersenne Twister algorithm. The number should be an unsigned integer from 0 to 4294967295 (0 to 2^32-1).

  • BOX -- An integer specifying the binomial field box size in MPc. If the box size is X, the random 3-vectors will be distributed betweeen (0, 0, 0) and (X, X, X).

  • POINTS -- Number of points to generate. The number should be an unsigned integer from 0 to 4294967295 (0 to 2^32-1).

FILE FORMATS

All the files are to be in ASCII format, with numerical fields separated by whitespaces and newlines for human readability.

OUTPUT

The output file should have the following format, compatible with the correlator and compcorr programs:

        number of points (N)
        data low bound (3 components, separated by whitespace)
        data high bound (3 components, separated by whitespace)
        point 1 (3 components, separated by whitespace)
        .
        .
        .
        point N (3 components, separated by whitespace)
